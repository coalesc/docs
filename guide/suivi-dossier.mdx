## Vue d'ensemble

Chaque mandat affiche une barre de progression qui montre combien de documents requis ont été reçus.

- **Barre grise**: aucun document reçu
- **Barre noire**: documents en cours de réception
- **Barre verte**: tous les documents requis sont reçus, le dossier est prêt

Le compteur dans l'onglet **Liste de documents** affiche le ratio documents reçus / documents requis (ex: 3/8).

<Info>
Seuls les documents **requis** comptent dans la progression. Les documents optionnels n'empêchent pas le dossier d'être considéré complet.
</Info>

## Statuts des documents

| Statut | Signification |
|--------|---------------|
| **Manquant** | Document requis, pas encore reçu |
| **Optionnel** | Document souhaité mais pas obligatoire |
| **Reçu** | Document reçu et classifié |

## Statuts du mandat

| Statut | Signification |
|--------|---------------|
| **Non commencé** | Le mandat vient d'être créé |
| **En cours** | Des documents sont en cours de réception |
| **Prêt** | Tous les documents requis sont reçus |
| **Complété** | Le travail comptable est terminé |

## Filtrer les mandats

Dans la liste des mandats, vous pouvez filtrer par:

- Statut (non commencé, en cours, prêt, complété)
- Type de mandat (T1, T2, tenue de livres, etc.)
- Client

## Historique

Chaque mandat garde un historique complet:

- Quand les documents ont été reçus
- Par quel canal (courriel, portail client, upload manuel)
- Quand les relances ont été envoyées

Cliquez sur un document reçu pour le prévisualiser directement dans Coalesc.
