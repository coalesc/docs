## Qu'est-ce qu'un questionnaire?

Un questionnaire est un formulaire intelligent envoyé à votre client. Au lieu de lui demander "Avez-vous un T4?", le questionnaire demande "Êtes-vous employé?". Les réponses génèrent automatiquement la liste de documents personnalisée.

## Comment ça fonctionne

<Steps>
  <Step title="Envoi">
    Dans un mandat, cliquez sur **Envoyer le questionnaire**. Le client reçoit un courriel avec un lien unique.
  </Step>
  <Step title="Réponses">
    Le client répond à des questions simples, une à la fois (expérience conversationnelle). Les questions s'adaptent selon les réponses précédentes.
  </Step>
  <Step title="Checklist générée">
    Les réponses génèrent automatiquement la liste de documents que le client doit fournir. Un client sans enfants ne verra pas les frais de garde.
  </Step>
  <Step title="Upload">
    Le client voit sa liste personnalisée et peut directement déposer ses documents.
  </Step>
</Steps>

## Types de questions

| Type | Exemple |
|------|---------|
| **Oui / Non** | Êtes-vous employé? |
| **Choix unique** | Quel est votre état civil? |
| **Choix multiple** | Quelles déductions s'appliquent? |
| **Numérique** | Combien d'employeurs avez-vous eu? |
| **Information** | Texte explicatif (pas de réponse requise) |

## Questionnaires par type de mandat

Chaque type de mandat peut avoir son propre questionnaire:

- **T1**: Questions sur la situation personnelle, emploi, revenus, déductions
- **T2**: Questions sur la structure corporative, revenus, dépenses
- **Tenue de livres**: Questions sur les comptes bancaires, fournisseurs, paie

## Créer un questionnaire

1. Allez dans **Questionnaires** dans le menu
2. Cliquez sur **Nouveau questionnaire**
3. Choisissez le type de mandat
4. Construisez l'arbre de questions avec le constructeur visuel

## Questionnaire par défaut vs personnalisé

Coalesc fournit des questionnaires par défaut pour T1 et T2. Vous pouvez les utiliser tels quels ou créer les vôtres.

<Tip>
Les questions sont bilingues (français et anglais). Le client voit le questionnaire dans sa langue préférée.
</Tip>
